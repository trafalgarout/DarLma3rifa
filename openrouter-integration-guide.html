دليل تكامل OpenRouter مع دار المعرفة

هذا الدليل يشرح كيفية استبدال الدردشة التجريبية بدردشة حقيقية مدعومة بواسطة OpenRouter.

ما هو OpenRouter؟

OpenRouter هي خدمة تتيح لك الوصول إلى مجموعة متنوعة من نماذج الذكاء الاصطناعي من خلال واجهة برمجة تطبيقات (API) موحدة. تقدم OpenRouter خطة مجانية تتيح لك استخدام بعض النماذج مثل Deepseek بدون تكلفة.

المتطلبات الأساسية

  1. حساب على OpenRouter
  2. مفتاح API من لوحة تحكم OpenRouter
  3. ملفات JavaScript التي قمنا بإنشائها:
    • openrouter-chat.js - تنفيذ الدردشة
    • chatbot-config.js - إعدادات الرفقاء المختلفين

خطوات التنفيذ

  1. إنشاء حساب في OpenRouter

    قم بزيارة OpenRouter.ai وإنشاء حساب جديد. بعد التسجيل، انتقل إلى لوحة التحكم للحصول على مفتاح API الخاص بك.

  2. إضافة ملفات JavaScript إلى صفحات الرفقاء

    قم بإضافة الأسطر التالية قبل نهاية وسم </body> في كل صفحة من صفحات الرفقاء:

    <!-- OpenRouter Chatbot Integration -->
<script src="js/openrouter-chat.js"></script>
<script src="js/chatbot-config.js"></script>
<script>
    // استبدل 'YOUR_OPENROUTER_API_KEY' بمفتاح API الخاص بك
    const openRouterApiKey = 'YOUR_OPENROUTER_API_KEY';
    
    // تهيئة الدردشة
    document.addEventListener('DOMContentLoaded', function() {
        initializeOpenRouterChatbot(openRouterApiKey);
    });
</script>

  3. تخصيص إعدادات الرفقاء (اختياري)

    يمكنك تخصيص إعدادات كل رفيق في ملف chatbot-config.js. لقد قمنا بالفعل بإعداد التكوينات الأساسية لجميع الرفقاء.

  4. اختيار نموذج الذكاء الاصطناعي

    يمكنك اختيار نموذج مختلف من OpenRouter عن طريق تعديل خاصية model في ملف openrouter-chat.js. النموذج الافتراضي هو deepseek/deepseek-coder الذي يقدم رصيداً مجانياً.

    بعض النماذج المجانية الأخرى التي يمكنك تجربتها:

    • anthropic/claude-instant-1
    • google/gemini-pro
    • mistralai/mistral-7b-instruct

    يمكنك الاطلاع على قائمة كاملة بالنماذج المتاحة في توثيق OpenRouter.

  5. اختبار التكامل

    بعد إضافة الكود إلى صفحات الرفقاء، قم بزيارة إحدى الصفحات واختبار الدردشة للتأكد من أنها تعمل بشكل صحيح.

مثال على التنفيذ

لقد قمنا بإنشاء مثال على التنفيذ في صفحة business-thinker.html. يمكنك استخدام هذه الصفحة كمرجع لتنفيذ التكامل في الصفحات الأخرى.

ملاحظة مهمة: لا تقم بمشاركة مفتاح API الخاص بك علناً أو تضمينه في الكود المصدري المنشور. في بيئة الإنتاج، يجب تخزين مفتاح API في متغير بيئة أو خدمة آمنة.

حدود الاستخدام المجاني

تقدم OpenRouter حدوداً للاستخدام المجاني تختلف حسب النموذج المستخدم. تأكد من مراجعة صفحة التسعير للحصول على أحدث المعلومات حول الحدود المجانية.
